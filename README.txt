== Description ==

This plugin add a Widget on frontend to get rating of use experience of site visiting.

### Features:
* Add Widget to site for experience of site visit.

== Installation ==

1. Upload the plugin files to the plugin directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Rate the Site experience" plugin through the 'Plugins' screen in WordPress
3. Click on "CV Rate the Site experience" on the dashboard.

== Frequently Asked Questions ==

= How to setup Plugin? =

Activate the plugin and Make changes as per requirement from plugin setting dashboard.
